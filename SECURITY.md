# Security Policy

## Supported Versions

Only the latest minor release is supported for security updates.

## Reporting a Vulnerability

Please report vulnerabilities privately via the repository security advisory workflow or by contacting the maintainers directly.

Include:

- Version and platform
- Reproduction steps
- Impact assessment
- Any proof-of-concept code

## Response Targets

- Initial acknowledgment: within 48 hours
- Triage decision: within 5 business days
- Patch timeline: based on severity and exploitability

## Disclosure

We follow coordinated disclosure. Please do not publish details before a fix is available and users have time to upgrade.
